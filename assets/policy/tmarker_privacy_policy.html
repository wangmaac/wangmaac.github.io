Privacy Policy

TMarker는 사용자의 개인정보를 수집하거나 서버로 전송하지 않습니다. 이 페이지는 TMarker의 데이터 처리 방식, 접근성 권한 사용 목적, 로컬 설정 저장 범위를 설명합니다.

시행일: 2026년 4월 2일
Effective date: April 2, 2026

한국어

1. 수집하는 개인정보

TMarker(이하 "앱")는 사용자의 개인정보를 수집하거나 서버로 전송하지 않습니다.

앱은 회원가입, 로그인, 연락처, 위치, 결제정보, 광고 식별자, 분석 데이터, 추적 데이터를 수집하지 않습니다.

한국어

2. 앱이 처리하는 정보

앱은 기능 제공을 위해 다음 정보를 사용자 기기 내부에서만 처리할 수 있습니다.

  • 입력 소스 상태 표시를 위한 시스템 입력 소스 정보
  • 활성 윈도우 표시 기능 제공을 위한 접근성 권한 정보
  • 색상, 표시 모드, 효과, 표시 시간 등의 사용자 설정 값

이 정보는 사용자 기기 내부에서만 사용되며, 개발자 서버 또는 제3자에게 전송되지 않습니다.

한국어

3. 접근성 권한 사용

앱은 "활성 윈도우 표시" 기능을 제공하기 위해 macOS 접근성 권한을 요청할 수 있습니다.

이 권한은 현재 포커스된 창의 위치와 크기를 확인하여 해당 창 외곽선을 표시하기 위한 용도로만 사용됩니다. 앱은 사용자의 입력 내용, 문서 내용, 화면 내용 자체를 읽거나 저장하거나 전송하지 않습니다.
한국어

4. 로컬 저장 정보

앱은 macOS의 로컬 저장소(UserDefaults)를 사용해 아래 설정을 기기에 저장할 수 있습니다.

  • 입력 소스별 테두리 색상
  • 표시 모드
  • 효과 타입
  • 표시 시간

이 정보는 사용자 기기에만 저장되며 외부로 전송되지 않습니다.

한국어

5. 제3자 제공 및 추적

앱은 개인정보를 제3자에게 제공하지 않으며, 외부 서버로 전송하지 않습니다.

앱은 광고 SDK, 분석 도구, 사용자 추적 도구를 포함하지 않습니다.

한국어

6. 보관 및 삭제

앱은 개인정보를 수집하지 않으므로 별도로 보관하거나 파기할 개인정보가 없습니다.

사용자가 저장한 설정 값은 앱 삭제 또는 설정 초기화를 통해 기기에서 제거할 수 있습니다.

English

1. Personal Data Collected

TMarker does not collect personal data and does not transmit user data to any server.

The app does not collect account information, contact details, location data, payment information, advertising identifiers, analytics data, or tracking data.

English

2. Information Processed on Device

The app may process the following information locally on the user's device in order to provide its features:

  • System input source information used to display the current input status
  • Accessibility permission status used for the active window highlight feature
  • User preferences such as colors, display mode, effect type, and display duration

This information is processed only on the device and is not sent to the developer or to third parties.

English

3. Accessibility Permission

TMarker may request macOS Accessibility permission to support the "active window" display mode.

This permission is used only to detect the position and size of the focused window so the app can draw a highlight border around it. TMarker does not read, store, or transmit the content of what you type, your documents, or your screen.
English

4. Local Storage

The app may store settings locally using macOS UserDefaults, including:

  • Border colors by input source
  • Display mode
  • Effect type
  • Display duration

These settings remain on the device and are not transmitted externally.

English

5. Third Parties and Tracking

The app does not share personal data with third parties.

The app does not include advertising SDKs, analytics SDKs, or user tracking technologies.

English

6. Retention and Deletion

Because the app does not collect personal data, there is no separately retained personal information on a server.

Locally stored settings can be removed by deleting the app or resetting its preferences.
